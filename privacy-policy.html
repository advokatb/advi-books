Политика конфиденциальности

1. Введение

Добро пожаловать на сайт "Мой Книжный путь на LiveLib". Настоящая Политика конфиденциальности описывает, как мы собираем, используем и защищаем информацию, связанную с использованием нашего сайта.

2. Сбор данных

Мы собираем только данные, необходимые для предоставления функциональности сайта:

  • Имя пользователя LiveLib: Используется для получения данных о ваших книгах через публичный API LiveLib.
  • Локальное хранилище (localStorage): Сохраняем кэшированные данные о книгах для улучшения производительности.

3. Использование данных

Ваши данные используются исключительно для:

  • Отображения списка прочитанных, текущих и запланированных книг.
  • Создания статистики чтения и визуализации (графики, челленджи).
  • Кэширования данных для быстрого доступа при повторных посещениях.

4. Передача данных

Мы не передаем ваши данные третьим лицам, за исключением публичного API LiveLib, который используется для получения данных о книгах.

5. Хранение и удаление данных

Данные в localStorage хранятся на вашем устройстве и удаляются автоматически раз в день или при очистке кэша браузера. Вы можете вручную очистить localStorage через настройки браузера.

6. Контакты

Если у вас есть вопросы о нашей политике конфиденциальности, свяжитесь с нами через GitHub.

7. Изменения политики

Мы можем обновлять эту политику. Последняя версия всегда доступна по адресу https://advokatb.github.io/advi-books/privacy-policy.html.
